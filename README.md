# Portfolio Web Application

Aplikasi web portofolio ini dibangun menggunakan:
- Python Flask (Backend)
- HTML, CSS, JS (Frontend)
- TiDB (Online Database MySQL Compatible)
- Cloudinary (Image Upload)
- Resend (Email API)

## Setup Instalsi
1. Buka folder ini di terminal.
2. Buat virtual environment: `python -m venv venv`
3. Aktifkan venv: `venv\Scripts\activate` (Windows)
4. Install requirements: `pip install -r requirements.txt`
5. Konfigurasi file `.env` berdasarkan file `.env.example`.
6. Jalankan aplikasi: `python app.py`
